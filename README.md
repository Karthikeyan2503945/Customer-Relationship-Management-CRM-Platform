# 🚀 CRM360 – SaaS Customer Relationship Management Platform

CRM360 is a modern **full-stack CRM (Customer Relationship Management)** application built with the **MERN Stack**. It enables organizations to efficiently manage customers, leads, sales pipelines, tasks, reports, and user roles through a secure and scalable web application.

## ✨ Features

- 🔐 JWT Authentication & Role-Based Access Control (RBAC)
- 👥 Customer Management (CRUD)
- 🎯 Lead Management & Sales Pipeline
- 📊 Interactive Dashboard & Analytics
- 📅 Task & Activity Management
- 🔔 Real-time Notifications (Socket.io)
- 📈 Reports & Business Insights
- 👤 User Profile & Settings
- 🌙 Responsive UI with Dark/Light Mode
- 🔒 Secure REST API with Express & MongoDB

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/<your-username>/CRM360.git
cd CRM360
```

### Install dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd ../server
npm install
```

### Start Development

Backend

```bash
cd server
npm run dev
```

Frontend

```bash
cd client
npm run dev
```

Open:

```
http://localhost:5173
```
